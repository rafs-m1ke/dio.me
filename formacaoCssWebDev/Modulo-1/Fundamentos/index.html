Agrupamento de Seletores

Podemos aplicar as mesmas regras CSS a vários seletores, separando-os por vírgula.

Assim, não precisamos repetir as mesmas regras para cada seletor. Podemos dazer um só bloco de CSS reutilizar em vários seletores.
